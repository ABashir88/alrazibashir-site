# alrazibashir-site
Personal website and professional positioning site for Alrazi Bashir — Enterprise SaaS Account Executive specializing in AI, CX infrastructure, and conversational platforms.

## Tech stack

- **Framework**: Next.js 16
- **Language**: TypeScript
- **Styling**: Tailwind CSS 4

## Getting started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Then open `http://localhost:3000` in your browser.
